    .util.converge¶

    Converge.

    • See Converge.

    Parameters:

    Name Type Description
    func function | list | dict A unary function, or a list, or a dictionary.
    param * Parameter to the function, or index of the list, or keys of the dictionary.

    Returns:

    Type Description
    * param and successive evaluations by func, until two successive evaluations match or an evaluation matches param.

    .util.defined¶

    Whether a name is defined.

    • See key.

    Parameter:

    Name Type Description
    name symbol A name.

    Returns:

    Type Description
    table 1b if the name is already defined, 0b otherwise.

    .util.overBinary¶

    Over binary.

    • See Over.

    Parameters:

    Name Type Description
    func fn A binary function.
    start * A value.
    list *[] A list.

    Returns:

    Type Description
    *[] A list of results by successively apply the function to the elements in the list. The first iteration uses start and the first element of the list.

    .util.overTernary¶

    Over ternary.

    • See Over.

    Parameters:

    Name Type Description
    func fn A binary function.
    start * A value.
    list1 *[] A list.
    list2 *[] A list.

    Returns:

    Type Description
    *[] A list of results by successively apply the function to the elements in the list. The first iteration uses start, the first element of list1, and the first element of list2.

    .util.overUnary¶

    Over unary.

    • See Over.

    Parameters:

    Name Type Description
    func fn A binary function.
    list *[] A list.

    Returns:

    Type Description
    *[] A list of results by successively apply the function to the elements in the list.

    .util.trap¶

    Trap.

    • See Trap.

    Parameters:

    Name Type Description
    func fn A function.
    param * Parameter(s) to the function.
    expr * An expression or a unary function.

    Returns:

    Type Description
    * The result of evaluating the function with the parameter, if no error occurs. When there is an error, if expr is an expression, the expression is evaluated and returned; if expr is a function, it's evaluated given the error as the input.
